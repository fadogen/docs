## Qu'est-ce que Fadogen ?

**Fadogen** est une application macOS native, gratuite et open-source, qui vous permet de **créer, développer, déployer et exploiter** des applications PHP depuis un seul endroit.

Elle remplace une chaîne d'outils fragmentée par un workflow cohérent :

- pas de Homebrew
- pas de configuration Docker personnalisée
- pas de plateformes de déploiement payantes
- pas d'identifiants stockés sur des serveurs tiers

Installez simplement Fadogen, décrivez votre projet et déployez.

---

## Un workflow complet, de bout en bout

Fadogen couvre le **cycle de vie complet** de votre application :

1. Générez un projet Laravel ou Symfony
2. Développez en local avec HTTPS et des runtimes isolés
3. Générez une infrastructure et un CI/CD prêts pour la production
4. Provisionnez des serveurs et déployez sans interruption
5. Sauvegardez et restaurez les données automatiquement

Le tout depuis la même application.

## À qui s'adresse Fadogen ?

Fadogen est conçu pour :

- **Les développeurs solo** qui veulent livrer plus vite
- **Les freelances** qui gèrent plusieurs projets clients
- **Les petites équipes** sans ingénieurs DevOps dédiés

Si vous en avez assez d'assembler environnements locaux, outils de déploiement et services payants, Fadogen vous offre une voie plus simple.

---

## Open source, sans verrouillage

Fadogen est entièrement **open source**.

Tous les fichiers générés sont standards et portables :

- Docker
- Docker Compose
- GitHub Actions
- Configurations d'infrastructure

Vous pouvez déployer n'importe où, tout modifier ou partir à tout moment.

Pas d'abonnement. Pas de fonctionnalités cachées.
