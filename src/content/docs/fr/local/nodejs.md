## Node.js

Cliquez sur l'icône **Rafraîchir** dans la barre d'outils pour vérifier les nouvelles versions.

### Installer une version

1. Allez dans **Node.js & Bun** dans la barre latérale
2. Cliquez sur **Installer** à côté de la version souhaitée

### Version par défaut

Pour changer la version par défaut :
1. Allez dans **Node.js & Bun**
2. Cliquez sur **Définir par défaut**

### Version par projet

Chaque projet peut utiliser une version Node.js différente via le fichier `.fadogen`.

## Bun

Bun est un runtime JavaScript moderne et rapide. Une seule version peut être installée à la fois.

### Installer Bun

1. Allez dans **Node.js & Bun**
2. Cliquez sur **Installer** à côté de Bun

### Utiliser Bun par projet

Vous pouvez choisir entre npm et Bun pour chaque projet.

## Intégration shell

Les commandes `node`, `npm`, `npx` et `bun` utilisent automatiquement la version configurée pour le projet en cours.
