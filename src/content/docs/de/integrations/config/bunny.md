Bunny bietet DNS-Verwaltung.

## Funktionen

- **DNS**: Verwalten Sie DNS-Zonen und -Einträge

## API-Schlüssel abrufen

1. Gehen Sie zum [Bunny Dashboard](https://dash.bunny.net/)
2. Klicken Sie auf Ihr Profilsymbol > **Kontoeinstellungen**
3. Gehen Sie zu **API**
4. Kopieren Sie den API-Schlüssel

## Zu Fadogen hinzufügen

1. Gehen Sie zu **Integrationen**
2. Klicken Sie auf **Integration hinzufügen**
3. Wählen Sie **Bunny**
4. Fügen Sie Ihren API-Schlüssel ein
5. Klicken Sie auf **Speichern**

## Bunny verwenden

### DNS-Verwaltung

Ihre Bunny DNS-Zonen sind automatisch verfügbar, wenn Sie eine Site mit einem Server verknüpfen.

:::note
Ihre Domains müssen Bunny-Nameserver verwenden. Aktualisieren Sie die Nameserver bei Ihrem Domain-Registrar, um auf Bunny zu verweisen.
:::
