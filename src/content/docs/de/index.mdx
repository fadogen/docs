## Was ist Fadogen?

**Fadogen** ist eine kostenlose, quelloffene und native macOS-Anwendung, mit der Sie PHP-Anwendungen an einem einzigen Ort **erstellen, entwickeln, bereitstellen und betreiben** können.

Sie ersetzt eine fragmentierte Toolchain durch einen einheitlichen Workflow:

- kein Homebrew
- keine individuelle Docker-Konfiguration
- keine kostenpflichtigen Bereitstellungsplattformen
- keine Anmeldedaten auf Drittanbieter-Servern

Installieren Sie einfach Fadogen, beschreiben Sie Ihr Projekt und liefern Sie aus.

---

## Ein vollständiger Workflow, von Anfang bis Ende

Fadogen deckt den **gesamten Lebenszyklus** Ihrer Anwendung ab:

1. Generieren Sie ein Laravel- oder Symfony-Projekt
2. Entwickeln Sie lokal mit HTTPS und isolierten Laufzeitumgebungen
3. Generieren Sie produktionsreife Infrastruktur und CI/CD
4. Stellen Sie Server bereit und deployen Sie ohne Ausfallzeit
5. Sichern und stellen Sie Daten automatisch wieder her

Alles aus derselben App.

## Für wen ist Fadogen?

Fadogen ist konzipiert für:

- **Solo-Entwickler**, die schneller ausliefern möchten
- **Freelancer**, die mehrere Kundenprojekte verwalten
- **Kleine Teams** ohne dedizierte DevOps-Ingenieure

Wenn Sie es leid sind, lokale Umgebungen, Bereitstellungstools und kostenpflichtige Dienste zusammenzustückeln, bietet Ihnen Fadogen einen einfacheren Weg.

---

## Open Source, kein Lock-in

Fadogen ist vollständig **Open Source**.

Alle generierten Dateien sind standardisiert und portabel:

- Docker
- Docker Compose
- GitHub Actions
- Infrastruktur-Konfigurationen

Sie können überall deployen, alles anpassen oder jederzeit wechseln.

Keine Abonnements. Keine versteckten Funktionen.
