## Node.js

Klicken Sie auf das **Aktualisieren**-Symbol in der Symbolleiste, um nach neuen Versionen zu suchen.

### Eine Version installieren

1. Gehen Sie zu **Node.js & Bun** in der Seitenleiste
2. Klicken Sie auf **Installieren** neben der gewünschten Version

### Standardversion

Um die Standardversion zu ändern:
1. Gehen Sie zu **Node.js & Bun**
2. Klicken Sie auf **Als Standard festlegen**

### Version pro Projekt

Jedes Projekt kann eine andere Node.js-Version über die `.fadogen`-Datei verwenden.

## Bun

Bun ist eine moderne, schnelle JavaScript-Laufzeitumgebung. Es kann jeweils nur eine Version installiert werden.

### Bun installieren

1. Gehen Sie zu **Node.js & Bun**
2. Klicken Sie auf **Installieren** neben Bun

### Bun pro Projekt verwenden

Sie können für jedes Projekt zwischen npm und Bun wählen.

## Shell-Integration

Die Befehle `node`, `npm`, `npx` und `bun` verwenden automatisch die für das aktuelle Projekt konfigurierte Version.
