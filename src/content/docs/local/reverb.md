[Laravel Reverb](https://laravel.com/docs/reverb) is a WebSocket server for real-time applications.

Click the **Refresh** icon in the toolbar to check for new versions.

## Installation

1. Go to **Reverb** in the sidebar
2. Click **Install**

## Configuration

### Port

Default port: `8080`

Modifiable if needed.

### Environment Variables

```bash
BROADCAST_DRIVER=reverb
REVERB_APP_ID=1001
REVERB_APP_KEY=laravel-fadogen
REVERB_APP_SECRET=secret
REVERB_HOST="reverb.localhost"
REVERB_PORT=443
REVERB_SCHEME=https
```

Click the **Connection** icon in the toolbar to view credentials.

## Auto-Start

Enable auto-start to launch Reverb when Fadogen starts.

## Data

Binaries are installed in `/Users/Shared/Fadogen` (shared across Mac accounts).
