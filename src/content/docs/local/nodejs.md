## Node.js

Click the **Refresh** icon in the toolbar to check for new versions.

### Install a Version

1. Go to **Node.js & Bun** in the sidebar
2. Click **Install** next to the desired version

### Default Version

To change the default version:
1. Go to **Node.js & Bun**
2. Click **Set as Default**

### Per-Project Version

Each project can use a different Node.js version via the `.fadogen` file.

## Bun

Bun is a modern, fast JavaScript runtime. Only one version can be installed at a time.

### Install Bun

1. Go to **Node.js & Bun**
2. Click **Install** next to Bun

### Use Bun Per Project

You can choose between npm and Bun for each project.

## Shell Integration

The `node`, `npm`, `npx`, and `bun` commands automatically use the version configured for the current project.
