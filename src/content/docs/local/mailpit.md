[Mailpit](https://github.com/axllent/mailpit) is a local email server that captures all outgoing emails from your applications.

## Web Interface

Access the Mailpit web interface at `https://mail.localhost` to view captured emails.

## SMTP Configuration

Click the **Connection** icon in the toolbar to view credentials.

- **Host**: `127.0.0.1`
- **Port**: `1025`
- **Username**: Empty
- **Password**: Empty
- **Encryption**: None
