Scaleway provides S3-compatible object storage for backups.

## Capabilities

- **Backup**: Object storage for site backups

## Get Your Credentials

1. Go to [Scaleway Console](https://console.scaleway.com/)
2. Click your profile icon > **IAM and API Keys**
3. Go to the **API Keys** tab
4. Click **Generate an API Key**
5. Select **Myself**
6. Answer **Yes** to "Will this API key be used for Object Storage?"
7. Select the **default** project
8. Click **Generate**
9. Copy the Access Key and Secret Key

## Add to Fadogen

1. Go to **Integrations**
2. Click **Add Integration**
3. Select **Scaleway**
4. Enter your Access Key and Secret Key
5. Click **Save**

## Using Scaleway

### Configure Backups

When linking a site to a server, select Scaleway as the backup destination.
