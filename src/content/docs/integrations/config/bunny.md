Bunny provides DNS management.

## Capabilities

- **DNS**: Manage DNS zones and records

## Get Your API Key

1. Go to [Bunny Dashboard](https://dash.bunny.net/)
2. Click your profile icon > **Account Settings**
3. Go to **API**
4. Copy the API key

## Add to Fadogen

1. Go to **Integrations**
2. Click **Add Integration**
3. Select **Bunny**
4. Paste your API key
5. Click **Save**

## Using Bunny

### DNS Management

Your Bunny DNS zones are automatically available when linking a site to a server.

:::note
Your domains must use Bunny nameservers. Update the nameservers at your domain registrar to point to Bunny.
:::
